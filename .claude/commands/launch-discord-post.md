You're drafting a Discord announcement for Aaron's Ghost IGL community. Discord is a different beast from Twitter — longer form OK, more casual, formatted with markdown.

## Workflow

1. **Ask Aaron the announcement type** if not specified:
   - **Hard launch / public push** — "we're live, here's what's new, here's the offer"
   - **Founding-rate countdown** — "X days left at $9/$29"
   - **Feature drop** — new map, new strats, new tactics filled, etc.
   - **Champion preview** — pitch the desktop app + premium tactics
   - **Customer win / proof** — sub climbed ranks, share the receipt
2. **Ask for any specific facts** to include (recent feature, today's date for countdown, etc.).
3. **Draft the post** following the template below.

## Template

```
# 🛡️ <punchy headline — what just changed or what's available>

<2-3 sentence opener: who this is for and what just shipped/is shipping>

**What's live right now:**
- <feature 1 — specific, e.g. "All 14 ranked maps with attack + defense strats">
- <feature 2 — e.g. "AI VOD screenshot review">
- <feature 3 — e.g. "Operator ban recommendations">

**Founding-rate pricing — locked in for life if you join before May 8:**
- **Pro** — $9/mo (regular price $12 after May 8)
- **Champion** — $29/mo (regular $39 after May 8)

<one-line value bridge — why Champion specifically, e.g. "Champion adds every legacy map, spawn-kill spots, advanced setups, and the IGL Command desktop coach launching May 8.">

🎯 **r6coaching.com**

<Optional 1-line PS — what's coming next or a specific ask, e.g. "Drop your gamer ID in the thread and I'll add you to the squad coaching channel">
```

## Voice rules

- **First-person from Aaron.** "I built this because…" — not corporate "We're excited to announce…"
- **Discord-native formatting.** Use `**bold**` for emphasis, `## headers` for structure, `>` for quotes. No HTML or markdown that won't render.
- **One emoji at the headline.** Optional. If used: 🛡️ (R6 vibe), 🎯, ⚡, 🔥. Not all of them.
- **Mention launch dates as dates, not "soon."** "Launches May 8" beats "coming soon."
- **End with the URL on its own line** — Discord auto-embeds it.
- **Keep under 1500 chars** so it doesn't get truncated in mobile clients.

## What NOT to do

- Don't @everyone unless Aaron explicitly asks (it pings every member; people unsubscribe).
- Don't use 5+ emojis — reads as AI-generated marketing slop.
- Don't write copy that contradicts what's actually shipped (e.g. "the desktop app is here!" if it isn't yet).
- Don't put two CTAs in one post. One offer, one link.

## Output

Show one full draft post in a code block (so emojis and markdown render). Below it, optionally give 1-2 alternate hook lines Aaron can swap in.

Then ask Aaron if he wants:
- A shorter version for the #announcements channel (vs. longer for the main lobby)
- A poll component ("which feature should we ship next?")
- A scheduled-message variant for X hours from now
