# Discord — Server Launch Post Template

## Goal

Get into existing FPS-game Discord servers and offer free Champion access to first N joiners as launch promo. Builds initial Discord member base + early adopters who become advocates.

## When to use

After the Recon+ Discord server is live and has at least:
- A clear server description
- A few channels (#general, #vod-uploads, #r6-strats, etc.)
- A bot or webhook ready to comp Champion accounts on join

## Posting strategy

**DON'T spam-post in 100 servers.** Pick the top 10 game-specific Discords (R6, CS2, Val, OW2, Apex, MVR, Halo, Finals, CoD, Fortnite community Discords) and reach out to mods FIRST.

## Mod approval message

Send this to the server's mod team via DM before posting:

```
Hi [server name] mods,

I run Recon+, an AI VOD review tool for [game]. Just launched our community Discord and would love to invite [game] players from your server.

I'd like to post a 1-time announcement offering free Champion access (normally $29/mo) to the first 50 of your members who join — no strings, just trying to seed the community with serious players.

If that's not allowed by your server rules, totally understand. If it is, I'd be happy to coordinate timing with you. The post would be brief, no spam follow-ups.

Thanks for considering,
Aaron — r6coaching.com
```

Wait for mod approval before posting. Most large servers will say yes for a 1-time community-building post; some will say no due to no-promo policies. Respect both answers.

## The post (after mod approval)

```
**Recon+ — Free Champion access for first 50 joiners**

Hey [server name],

Built an AI VOD review tool for [game]. After 6 months of working on it, just opened our community Discord and want to seed it with serious players from this server.

**The offer:**
- First 50 to join r6coaching.com Discord get free Champion access ($29/mo value, 30 days free, no card required)
- All 10 supported games unlocked
- No payment required, no follow-up sales spam — just the comp + a chat invite

**What it does:**
- Reads your replays, flags positioning + utility-timing patterns per round
- Tier-1 pro VOD comparisons for the patterns you have
- Tracks habits you don't realize you have (e.g., same anchor angle 4 rounds in a row)

**What it doesn't do:**
- Replace human coaching
- Train your aim
- Promise rank improvement

Discord invite: [your-invite-link]

Reply here or DM with questions. Mods, thanks for letting me post — happy to be removed if it doesn't fit the vibe.

— Aaron, r6coaching.com
```

## Variation per game

Adapt the framing per game:
- **R6:** "AI VOD review for R6 — flags Bandit-tricking timing, Mira angles, vertical spots."
- **CS2:** "AI VOD review for CS2 — flags smoke timing, anchor angles, AWP rotations."
- **Valorant:** "AI VOD review for Valorant — flags ult usage, agent positioning, lineup gaps."
- **OW2:** "AI VOD review for OW2 — flags ult chains, comp swaps, healing timing."
- (Etc. for each game.)

1. **R6:** Official Rainbow Six Siege Discord (1M+ members) — but may have strict no-promo rules. Try Coreross or RogueWall community Discord first.
2. **CS2:** Official Counter-Strike Discord, plus smaller "CS2 Coaching" or "CS2 Improvement" community Discords.
3. **Valorant:** Official Valorant Discord. Plus Valorant Coaching Discord if it exists.

6. **Marvel Rivals:** Official + r/marvelrivals Discord.
7. **Halo Infinite:** HCS Discord, Halo Improvement Discords.
8. **The Finals:** Official + r/thefinals Discord.
9. **CoD:** Official Warzone Discord, plus competitive sub-Discords.
10. **Fortnite:** Official + r/FortniteCompetitive Discord.

## Don't

- Don't post in more than 10 servers in the first month.
- Don't post twice in the same server within 3 months.
- Don't reply to "is this a scam?" with anything except the actual link to your About / pricing page.
- Don't argue with skeptics. "Free Champion, no card. Try it or don't."
- Don't post the same message in 5 channels of the same server. Mods nuke this.

## Track responses

Spreadsheet columns:
- Server name
- Mod DM date
- Approval Y/N
- Post date
- Joiners attributed (use unique invite link per server)
- Conversions (comped → paid)

## Discord-specific etiquette

- **Don't @everyone or @here.** Instant ban.
- **Don't create roles or push channel changes.** Just post and engage.
- **Use server-appropriate tone.** Some servers are casual; some are tactical. Match the energy.
- **Be ready to handle 50-200 sign-ups in 24 hours** if the post hits well. Have your comp automation ready before posting.
