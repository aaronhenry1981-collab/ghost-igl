Site-By-Site Notes

A Site (Heaven / Default) —

Attack: Standard A Site execute: Astra smokes the CT cross + Heaven, Sova flashes/recons over the choke, Jett entries first contact while Cypher watches the off-site flank. Plant default for the post-plant util cross. A Long control wins — dash + flash for the Heaven trade is the textbook A take.

Defense: 1-2-2 default: solo Mid for info, two on A Site (anchor + trade), two on A Heaven. Cypher sets traps cross-site. Astra flexes. Listen for the Mid take — that's the rotate trigger. Heaven anchor with A Long Sage wall is the standard hold; rotate from C through Garage.

B Site (Mid) —

Attack: Mid takeover: Sova recons Mid on round-start, Jett dashes the pick attempt, Astra smokes the rotate. Once mid is locked, split-execute B Site with Cypher pinching from off-site. Mid double-flash isolates B; pinch from A Link or C Garage on the rotate.

Defense: Eco / save round: stack B Site with all 5, save ults. Cypher traps the entry choke, Jett trades. Goal is a round win or clean save — do not chase frags into the rotate. Mid Doors trips + B Default anchor is the textbook hold; save ult for retake.

C Site (Long / Lobby) —

Attack: Default → re-fake: take map control through Mid and C Long, force the rotate, then re-fake to C Site. Astra smokes only after defenders bite on the off-site noise. Jett entries late. C Long take with Sage wall blocks Garage rotation; Phoenix wall enables free entry.

Defense: Aggressive Mid setup: Jett contests early with Sova support, Cypher anchors C Site solo. Astra flexes on commit. Goal is a round-opener pick that flips utility plan. C Long molly cycle + Garage rotate is mandatory; do not solo-anchor C on commit.
